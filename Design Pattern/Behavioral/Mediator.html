JavaScript Design pattern:
  • Creational - Mediator
    • Define an object that encapsulate how set of objects are interacting, Take for example a page in which you enter options to make a flight reservation. A simple Mediator rule would be: you must enter a valid departure date, a valid return date, the return date must be after the departure date, a valid departure airport, a valid arrival airport, a valid number of travelers, and only then the Search button can be activated.
    • The Mediator pattern provides central authority over a group of objects by encapsulating how these objects interact.
